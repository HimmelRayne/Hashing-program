Buatkan program lengkap menggunakan bahasa Java dan Python yang mengimplementasikan Hash Table untuk menyimpan data numerik (code bersih tanpa comment) dengan spesifikasi berikut:

Fitur program:

1. Input data
2. Cari data
3. Hapus data
4. Tampilkan hash table
5. Exit

Ketentuan program:

* Saat program pertama kali dijalankan, hash table harus otomatis terisi 100 angka random unik (tidak boleh ada angka yang sama).

* Gunakan fungsi hash sederhana namun benar untuk:

  * Primary hash function
  * Secondary hash function
* Program wajib menggunakan implementasi hash table manual, tidak boleh memakai built-in dictionary, hashmap, set, atau collection sejenis sebagai struktur utama.
* Tampilkan proses probing saat collision terjadi agar mudah dipahami.
* Buat menu interaktif berbasis terminal/console.
* Gunakan clean code dan struktur program yang rapi seperti standar senior programmer.
* Pisahkan fungsi-fungsi penting seperti:

  * insert()
  * delete()
  * search()
  * display()
  * primaryHash()
  * secondaryHash()
* Berikan validasi input yang baik.
* Gunakan ukuran table yang ideal dan jelaskan alasan pemilihannya.
* Berikan komentar penjelasan pada bagian logika penting, khususnya pada proses double hashing.
* Setelah kode selesai, jelaskan:

  1. Cara kerja program
  2. Cara kerja double hashing
